# Crawl4LeetCode
LeetCode 爬虫的设计初衷为个人学习与数据整理（需明确：未经 LeetCode 官方授权，任何商业用途或大规模数据爬取均属违规），常见合理用途包括：  题目信息结构化整理：获取公开题目库的基础信息（如题目 ID、标题、难度、标签、题干描述、输入输出示例），构建个人题解笔记库或按算法类型分类的刷题清单，生成为json文档
